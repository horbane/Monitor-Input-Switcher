# Upload Checklist

Before uploading this folder to GitHub, check:

- [ ] No `config.json`.
- [ ] No config backup files.
- [ ] No personal paths like `C:\Users\<your-name>`.
- [ ] No secrets or tokens.
- [ ] `README.md` looks correct.
- [ ] NirSoft / ControlMyMonitor notice is included.
- [ ] Screenshots do not show private information.
- [ ] Release files are uploaded separately under GitHub Releases, not committed to source if unnecessary.

Notes:

- Installer `.exe` files should usually go in GitHub Releases, not the source repository.
- Portable ZIP files should usually go in GitHub Releases, not the source repository.
- ControlMyMonitor is third-party software by NirSoft / Nir Sofer. Include it in the repository only if you are comfortable with its redistribution terms.
