## ControlMyMonitor

Monitor Input Switcher uses ControlMyMonitor by NirSoft / Nir Sofer as the monitor-control backend.

ControlMyMonitor is used to read and set monitor DDC/CI VCP values, including VCP code 60 for input source switching.

If included in a release package, ControlMyMonitor is distributed unmodified.
